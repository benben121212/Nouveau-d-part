Bienvenue sur ma page Test

Le HTML est un langage de balisage utilisé pour créer des pages web. Les balises HTML sont des éléments qui définissent la structure et le contenu de la page web. Les balises sont composées d'un nom et d'attributs qui spécifient des informations supplémentaires sur l'élément. Par exemple, la balise <img> est utilisée pour insérer des images dans la page et a des attributs tels que "src" pour spécifier l'emplacement de l'image et "alt" pour fournir une description alternative pour les utilisateurs qui ne peuvent pas voir l'image. Le code HTML est généralement écrit à l'aide dun éditeur de texte comme Sublime Text ou Visual Studio Code.

Les feuilles de style en cascade (CSS) sont utilisées pour définir la présentation et la mise en page des pages web. Les règles CSS sont appliquées aux éléments HTML en utilisant des sélecteurs et des propriétés. Par exemple, la règle "color: red;" applique la couleur rouge au texte sélectionné.

Les langages de programmation tels que JavaScript sont utilisés pour ajouter de l'interactivité aux pages web. Les scripts JavaScript peuvent être utilisés pour manipuler le contenu de la page, répondre aux actions de l'utilisateur et communiquer avec les serveurs.
